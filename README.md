# youframe

[![Play demo on youtube](https://img.youtube.com/vi/ERw_3MKKbOg/0.jpg)](https://youtu.be/ERw_3MKKbOg)

## Instructions to run code
0. `git clone https://github.com/ramsuthar305/youframe`
1. `cd server`
2. `npm install`
3. `npm run dev`
4. open new terminal
5. `cd client`
6. `npm install`
7. `npm start`

## Tech stack used
1. mongodb Atlas
2. React.js
3. Express.js
